### 请使用一个div写出有三条横线的小图标

::: demo 上下两条横线由`border`设置。中间的横线是设置的高度。将`background-clip`设置为`content-box`。也就是中间横线的高度。再用`padding`调整间距即可

```html
<template>
  <div class="menu"></div>
</template>

<style>
.menu {
  margin:0 auto;
  width: 30px;
  height: 5px;
  border-top: 5px solid red;
  border-bottom: 5px solid red;
  background-color: red;
  padding: 5px 0;
  background-clip: content-box;
}
</style>

```
