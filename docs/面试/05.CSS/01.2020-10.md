## 使用css实现模糊的效果

:::

::: demo 使用`filter`的`blur`属性

```html
<template>
<div class='demo-container'>
  <div class="blur">要模糊的内容</div>
</div>
</template>

<style>
.demo-container .blur{
  filter:blur(1px)
}
</style>
```
