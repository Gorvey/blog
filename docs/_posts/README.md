### 软件可靠性和硬件可靠性的区别(2016)

1. 最明显的硬件有老化磨损，硬件失效。软件没有
2. 硬件可靠性与`时间`，`设计，生产，运用`的过程有关。软件可靠性与`数据`有关。更多看人
3. 硬件纠错可以修复或更换。软件只能重新设计
4. 硬件可以预防。可以断开失效的部件诊断故障
